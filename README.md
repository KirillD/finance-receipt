For dev Environment:

- Install project depencies

`composer install`

- Reloads DB, updates table fields

`bin/console d:d:cr`

`bin/console d:mi:mi`

- Load fixtures data

`bin/console d:f:l`

- start server with 

`bin/console s:start`

- Go to 

`/api/doc`

- Run tests

`vendor/bin/phpunit`

Have fun
